#JSON.stringify()- это метод в JavaScript, который используется для преобразования объекта JavaScript в строку в формате JSON.
#Этот метод принимает объект в качестве аргумента и возвращает его строковое представление в формате JSON. Вот пример использования JSON.stringify();


#Этот метод полезен, когда вам необходимо передать данные между клиентом и сервером, сохранить данные в локальном хранилище
#браузера или в других сценариях, где требуется работа с данными в формате JSON. Обратная операция, 
#то есть преобразование JSON-строки обратно в объект JavaScript, выполняется с использованием метода JSON.parse().
